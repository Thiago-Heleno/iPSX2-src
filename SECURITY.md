# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest main | ✅ |

## Reporting a Vulnerability

Please open a [GitHub Issue](../../issues) with the label `security`.

This project is a PS2 emulator sideloaded on iOS. It does **not** process untrusted remote content or handle user credentials. The primary security surface is the BIOS/ROM loading path.
